A Small Project

Posted on August 2, 2018

Pleased to announce for Jump Bike D.Cs open API, that I have created a wrapper for it here.

Simple to work with just initialize with for example query = JumpBikeAPI() and then all the possible feeds to call from are open to submit a GET request to.

Currently working on it to later submut to PyPi, but for now just copy the file and do from main import JumpBikeAPI, base_url to be able to make API calls.

The next step is to put this on PyPi, but also create wrappers for many other transport startup APIs, which do not have public wrappers yet.
